# 我穿越成了特不靠谱
## 第三卷：竞选风云
### 第040章 支持者的困惑与狂热

俄亥俄州，特不靠谱竞选集会现场。

五万名支持者挤满了体育场，他们挥舞着"MAGA"的帽子，高喊"双重伟大"的口号。现场气氛热烈得像是要爆炸。

后台，李睿正在准备演讲稿。

"让我来！"特不靠谱在脑海里喊道，"这种场合你最不擅长了！"

"好吧，"李睿妥协了，"但你记住，别说太过分的话，我们的民调刚掉了5个百分点。"

"知道了知道了！"

特不靠谱接管了身体，大踏步走上舞台。

"USA！USA！USA！"支持者们疯狂欢呼。

特不靠谱张开双臂，享受着这震耳欲聋的欢呼声。这是他最爱的时刻——被崇拜、被追随、被无条件地爱戴。

"谢谢你们！"他高声说，"谢谢你们来到这里！你们是最棒的！tremendous！"

欢呼声更大了。

"我知道，最近有很多人说我疯了，说我有两个人格，说我不知道自己在说什么！"

台下有人喊："我们相信你，老大！"

"但我要告诉你们——"特不靠谱的声音提高了，"他们说得对！"

台下瞬间安静了。

"我确实有两种想法！一种是理性的，一种是直觉的！一种是大脑的，一种是心脏的！"

支持者们面面相觑。

"这有什么错？"特不靠谱继续说，"你们难道没有犹豫过吗？没有纠结过吗？没有在心里跟自己辩论过吗？"

台下开始有人点头。

"我有！"特不靠谱指着自己的胸口，"我经常跟自己辩论！一个我说'要强硬'，另一个我说'要灵活'！一个我说'要战斗'，另一个我说'要谈判'！"

"这就是我的秘密武器！"他高声宣布，"双重伟大！Double Great！"

台下爆发出雷鸣般的掌声。

"其他人只有一个想法，一个立场，一成不变！但我有两个！我比他们都多一倍！这就是我为什么能赢！"

"双重伟大！双重伟大！"支持者们开始高喊。

李睿在脑海里目瞪口呆："他在干什么？"

"我在赢回人心！"特不靠谱得意地说。

"但你把'双重人格'说成是优势？"

"本来就是优势！"

台上，特不靠谱继续说："拜登那个老糊涂，他有什么？除了老年痴呆，什么都没有！他连一个完整的想法都没有！"

台下哄堂大笑。

"但我有两个！有时候三个！我脑子里全是想法！tremendous ideas！"

支持者们笑得更欢了。

"所以，当他们说我疯了的时候，我要说——是的，我是疯了！疯狂地想赢！疯狂地想为美国做事！"

"USA！USA！USA！"

集会的气氛达到了高潮。

但在台下的人群中，也有一些人露出了困惑的表情。

杰克是一名来自宾夕法尼亚州的工厂工人，他投了特不靠谱两次票。此刻，他皱着眉头对旁边的妻子说："亲爱的，你不觉得老大今天...有点奇怪吗？"

"哪里奇怪？"妻子问。

"他说他有两个人格...这听着像是承认自己精神分裂啊。"

"那不是精神分裂！"旁边的一个年轻支持者插嘴，"那是一种高级的思维方式！你没听他说吗？他比其他人多一倍的想法！"

"但是..."

"没有但是！"年轻人激动地说，"这就是特不靠谱伟大的地方！他不完美，他有挣扎，但他诚实！他敢于承认自己的内心冲突！"

杰克想了想，点点头："也许你是对的..."

但他的眼神里，依然有一丝不安。

在体育场的另一个角落，一位名叫玛丽的退休教师正在和朋友们讨论。

"我觉得他需要帮助，"玛丽说，"不是政治上的帮助，是心理上的。"

"你是说...看心理医生？"朋友问。

"对。你们没看新闻吗？心理学专家都说他可能有多重人格障碍。"

"那是假新闻！"另一个朋友反驳，"主流媒体都在抹黑他！"

"但这是他自己在集会上说的啊..."

"那是策略！他在反讽！"

玛丽摇摇头："我不知道...我觉得这不像策略。"

台上，特不靠谱的演讲进入了尾声。

"所以，我的朋友们，"他说，"不要被那些假新闻迷惑！我有两个想法，不是病，是天赋！我会用这两个想法，让美国变得前所未有的伟大！"

"双重伟大！双重伟大！"

"我们会赢！我们会狠狠地赢！我们会赢两次！因为——"

他停顿了一下，环顾四周。

"我有两个人格，但我只有一个目标：让美国再次伟大！MAGA！"

"MAGA！MAGA！MAGA！"

集会结束后，特不靠谱的支持者们带着复杂的情绪离开。

有些人更加狂热了，他们坚信特不靠谱的"双重人格"是一种超越常人的天赋。

有些人则开始动摇，他们不确定一个"精神分裂"的人是否适合当总统。

还有些人纯粹是被搞糊涂了——他们不知道该相信哪个特不靠谱。

而在海湖庄园，李睿和特不靠谱正在复盘。

"你疯了吗？"李睿在脑海里吼道，"你居然承认自己有两个想法？"

"那又怎样？"特不靠谱满不在乎，"反正他们已经这么猜了，不如顺水推舟，把它变成优势！"

"优势？这怎么可能是优势？"

"你看今天的支持率！"特不靠谱得意地说，"集会结束后，我们的线上捐款增加了200%！"

李睿一愣："真的？"

"真的！人们爱死这个了！他们说我'真实'、'人性化'、'敢于展示脆弱'！"

李睿无言以对。

也许...特不靠谱是对的？

在这个政治越来越表演化的时代，一个"精神分裂"的候选人，反而比那些完美无缺的政客更吸引人？

"好吧，"李睿妥协了，"但你以后能不能提前告诉我你的计划？"

"不能，"特不靠谱说，"我自己都不知道我下一秒要说什么！"

李睿扶额。

"我也是身不由己啊..."他喃喃自语。

当天晚上，福克斯新闻播出了集会的精彩片段。

评论员塔克·卡尔森兴奋地说："这是政治史上的革命性时刻！特不靠谱先生打破了一切规则，他不再假装自己是完美的，他展示了自己真实的内心挣扎！这比任何精心设计的竞选策略都更有效！"

而CNN的评论员则冷冷地说："这只是精神病患者在为自己的病情找借口。一个不能控制自己思想的人，怎么能控制一个国家？"

社交媒体上，两派观点激烈交锋。

#特不靠谱双重伟大# 和 #特不靠谱精神病# 同时上了热搜。

而在华盛顿，拜登看着这些数据，皱起眉头。

"没想到他能把这变成优势，"他对史密斯说，"看来我们需要调整策略了。"

"怎么调整？"

"既然他的'双重人格'成了卖点，我们就让它成为负担。"

"什么意思？"

拜登露出狡猾的笑容："我们要让两个特不靠谱互相打起来。"

（第040章 完）
