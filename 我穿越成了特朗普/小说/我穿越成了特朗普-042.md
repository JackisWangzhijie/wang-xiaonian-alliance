# 我穿越成了特不靠谱
## 第三卷：竞选风云
### 第041章 TikTok禁令风波

华盛顿特区，白宫椭圆形办公室。

虽然已经不再是总统，但李睿（特不靠谱）依然在这里保持着影响力。此刻，他正坐在办公桌前，面对着一群来自国会的议员。

"总统先生，"共和党参议员汤姆·科顿说，"关于TikTok的问题，我们需要您的表态。"

李睿还没来得及说话，特不靠谱就在脑海里喊："ban了它！立刻ban了！"

"等等，"李睿在心里说，"这个问题很复杂，不能这么简单处理。"

"有什么复杂的？那是中国的spy软件！ban了！"

"但上面有1.7亿美国用户，其中大部分是年轻人。如果我们ban了它，会失去很多年轻选民的支持。"

"选民？"特不靠谱嗤之以鼻，"我的选民不靠TikTok！"

"时代变了，唐纳德。"

台上的"特不靠谱"沉默了一会儿，表情在变化。

议员们面面相觑。

"总统先生？"科顿小心翼翼地问。

"我在思考，"李睿接管了控制权，"TikTok确实涉及国家安全问题，我们需要认真对待。但同时，我们也要考虑数百万美国用户的权益..."

"考虑个屁！"特不靠谱抢回控制权，"ban了它！那是中国的武器！"

"但..."

"ban了！"

台上的"特不靠谱"再次陷入自相矛盾的境地。

科顿困惑地问："总统先生，您的立场到底是什么？"

李睿奋力夺回控制权："我的立场是，我们需要一个全面的评估，然后..."

"没有评估！直接ban！"

"...制定一个负责任的方案。"

议员们交换了一个眼神。

"总统先生，"另一位议员说，"您能给我们一个明确的答案吗？是ban，还是不ban？"

"ban！"特不靠谱说。

"不ban，"李睿同时说。

然后"特不靠谱"摇了摇头，自言自语道："等等，我是想说..."

科顿忍不住问："总统先生，您...您还好吗？"

"我很好！"特不靠谱说。

"我有点头疼，"李睿同时说。

然后"特不靠谱"又摇了摇头。

会议室里一片尴尬的沉默。

最后，科顿咳嗽了一声："总统先生，要不...您先休息一下？我们改天再谈？"

"不！"特不靠谱说，"我现在就告诉你我的立场——TikTok必须被ban！"

"但是..."李睿试图阻止。

"没有但是！"特不靠谱转向议员们，"听着，我要你们推动一项法案，要求字节跳动在90天内出售TikTok，否则就ban了它！"

议员们一愣。

"90天？"科顿问，"这个期限会不会太短了？"

"不短！"特不靠谱说，"他们要么卖，要么滚！"

李睿在脑海里绝望地说："你知道这样做的后果吗？年轻人会恨死我们的！"

"恨就恨！"特不靠谱满不在乎，"反正他们也投不了几次票！"

"你疯了吗？18-29岁的选民有数千万！"

"那又怎样？"

"那意味着我们会输掉大选！"

特不靠谱沉默了一会儿，然后说："那就...给他们一个选择？"

"什么选择？"

"让他们选：要么支持ban TikTok的爱国者，要么支持TikTok的卖国贼！"

"这..."

李睿无语了。这个逻辑简直疯狂，但不知道为什么，他觉得特不靠谱可能又对了。

在这个极端分化的时代，简单的二元对立反而比复杂的平衡更容易赢得支持。

"好吧，"李睿妥协了，"但你得答应我一件事。"

"什么？"

"如果真要ban，给我们一个替代方案。比如...让美国公司收购它，变成一个'美国版TikTok'。"

"什么意思？"

"意思是，"李睿解释，"我们不ban TikTok本身，而是要求它变成美国的。这样既能解决安全问题，又不会得罪年轻用户。"

特不靠谱想了想："这听起来不错...但谁买得起？"

"马斯克？贝佐斯？扎克伯格？"

"扎克伯格？"特不靠谱嗤之以鼻，"那个蜥蜴人？"

"总比没有好。"

"好吧，"特不靠谱同意，"但ban的命令要我来下！"

"成交。"

台上，"特不靠谱"终于统一了口径："我的最终立场是——TikTok要么卖给美国公司，要么就滚出美国！90天期限，没有商量！"

议员们点点头，开始记录。

"但总统先生，"科顿问，"如果字节跳动拒绝出售呢？"

"那就ban了它！"特不靠谱说，"而且我要在集会上宣布这个消息，让所有人都知道，特不靠谱是保护美国安全的总统！"

李睿在脑海里叹气："又是这招..."

"这招管用！"

确实管用。

当天晚上，特不靠谱在推特上宣布："TikTok必须在90天内出售给美国公司，否则将被ban！国家安全不容妥协！"

这条推特立刻引爆了舆论。

年轻人们疯了。

#SaveTikTok 成为热搜第一，数千万用户在平台上抗议。

"特不靠谱要毁掉我们的生活！"

"没有TikTok我怎么活？"

"我要投票给拜登！"

但与此同时，保守派选民却欢呼雀跃。

"终于有人敢对中国强硬了！"

"保护我们的数据安全！"

"特不靠谱做得好！"

民调开始剧烈波动。

18-29岁年龄段的支持率暴跌15个百分点，但50岁以上年龄段的支持率上升了8个百分点。

"看到了吗？"特不靠谱得意地说，"我们赢得了真正的美国人！"

"但失去了年轻人，"李睿叹气，"他们会记住这一天的。"

"记住又怎样？等他们长大了就会明白我是对的！"

"问题是，"李睿说，"现在距离大选还有几个月，这些年轻人会投票的。"

特不靠谱沉默了一会儿，然后说："那就...给他们一个甜头？"

"什么甜头？"

"比如...减税？给学生减税？"

"学生本来就没多少税可交。"

"那...免费大学？"

"那是民主党的政策！"

"那就...免费的...呃..."

"汉堡？"李睿讽刺道。

"好主意！"特不靠谱兴奋地说，"每个大学生都可以免费吃麦当劳！"

"你在开玩笑吧？"

"没有！这叫'特不靠谱套餐'！每个美国大学生都可以凭学生证免费领一个汉堡！"

李睿彻底无语了。

但奇怪的是，当特不靠谱在第二天宣布"特不靠谱套餐"计划时，年轻人们的愤怒居然真的减少了一些。

"虽然ban了TikTok，但至少还有免费汉堡..."有人在社交媒体上写道。

"一个汉堡就想收买我们？"另一个人反驳。

"但那是免费汉堡啊..."

民调显示，年轻选民的支持率回升了3个百分点。

"看到了吗？"特不靠谱得意洋洋，"我的策略永远有效！"

李睿不得不承认，特不靠谱对选民的直觉有时候是惊人的准确。

但同时，他也开始担心——这种用短期利益换取长期支持的做法，真的可持续吗？

而在这场TikTok风波中，最大的赢家可能是...

"等等，"李睿突然想到一个问题，"如果真有人收购TikTok，你觉得会是谁？"

"不知道，"特不靠谱说，"反正不是我。"

"但如果你是总统，你可以影响这个决定..."

"所以呢？"

"所以，"李睿小心翼翼地说，"如果最后的结果是某个'合适'的人收购了TikTok，而你在竞选中得到了他的支持..."

特不靠谱沉默了一会儿，然后笑了："李睿，你比我想象的更会玩政治。"

"这不是玩政治，这是..."

"这是什么？"

李睿想了想，苦笑道："这也是身不由己啊。"

（第041章 完）
