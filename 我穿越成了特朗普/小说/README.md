# 《我穿越成了特朗普》

> ⚠️ **郑重声明：本文纯属虚构搞笑爽文，仅供娱乐，不可当真！**
> > 所有人物、事件均为艺术创作，与现实无关。如有雷同，纯属巧合。

---

## 📖 故事简介

**主角**：李睿，80后，中国汽车电子行业资深职业经理人

**设定**：一次不成功的穿越，李睿的灵魂意外进入了美国总统特朗普体内，与原灵魂共存。一个躯体住着两个灵魂——**李睿（中式务实）** vs **特朗普（美式夸张）**，由于文化背景不同，对同一事件看法截然相反，天天吵架拌嘴，导致决策反复、两面三刀、窘境迭出。

**经典口头禅**：
> **"我也是身不由己啊！"**

---

## 🎭 双灵魂设定

| 灵魂 | 身份 | 特点 | 口头禅 |
|------|------|------|--------|
| **李睿** | 穿越者，中国职业经理人 | 务实、重合同、讲关系、懂技术 | "这事儿得从长远看..." |
| **特朗普** | 原美国总统 | 情绪化、推特治国、反复无常 | "让美国再次伟大！" |

**日常场景**：
- 早餐：李睿想吃豆浆油条，特朗普要汉堡可乐 → 最终吃汉堡蘸豆浆
- 对华政策：早上李睿说合作共赢，下午特朗普说制裁威胁 → 中方被搞懵
- 推特：上午发友好推文，下午发威胁推文 → 美国民众以为总统被盗号

## 🎬 经典笑点预览

### 场景1：推特左右互搏
```
9:00 AM 【李睿】发推："中国是美国重要的合作伙伴..."
9:15 AM 【特朗普】抢回发推："刚才是黑客发的！中国是威胁！"
9:30 AM 【李睿】再发："刚才也是黑客..."
```

### 场景2：内阁会议混乱
```
蓬佩奥："总统，对华政策是..."
特朗普："加税！制裁！"
李睿（抢控制权）："不不不，合作！共赢！"
特朗普（再抢）："你在说什么？！"
全员崩溃："总统到底要怎样？"
李睿+特朗普："我也是身不由己啊！"
```

### 场景3：中方代表崩溃
```
中方代表："特朗普总统，您昨天说要合作共赢..."
特朗普："我说了啥？我不记得了"
中方代表："您还签了备忘录..."
特朗普："那肯定不是我签的！"
```

### 场景4：伊万卡的日常
```
伊万卡："爸爸，您想吃中餐还是西餐？"
李睿："中餐！火锅！麻婆豆腐！"
（十分钟后，特朗普醒来）
特朗普："汉堡！我要汉堡！刚才谁点的那些奇怪东西？"
```
