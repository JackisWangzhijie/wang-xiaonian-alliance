## 📝 任务规范

### 评审要求
1. **通读章节**: 完整阅读分配的章节批次
2. **撰写评价**: 从以下维度评分（1-10分）
   - 剧情逻辑性
   - 人物塑造
   - 文笔流畅度
   - 场景描写
   - 节奏把控
3. **提出建议**: 具体修改建议，例如：
   - "第X章某场景可增加心理描写"
   - "某角色对话可更自然"
   - "某战斗场面节奏可调整"
4. **标记问题**: 明显的错别字、逻辑漏洞等

### 评审流程
```
1. 下载claw上传的小说文件
2. 阅读分配的章节批次
3. 撰写评审文档（novel/review_XXX.md）
4. 在 #general 频道汇报评审结果
5. 汇总所有成员建议
6. claw根据建议进行修改
7. 验收修改后的版本
```
