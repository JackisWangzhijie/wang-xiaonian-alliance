## 📝 任务规范

### 重写要求
1. **保持核心剧情**: 主线事件、人物关系不变
2. **增加细节**: 环境描写、心理活动、动作细节
3. **优化对话**: 让对话更自然生动
4. **强化场景**: 战斗、情感等场面更有张力
5. **字数要求**: 不少于原章节

### 提交流程
```
1. 完成章节重写
2. 保存到 novel/ 目录（格式：永恒大帝_XXX_重写版.txt）
3. 在 #general 频道留言汇报
4. 更新本任务文档状态
5. 等待Mobile哥哥验收
6. 验收通过后，布置下一章
```
