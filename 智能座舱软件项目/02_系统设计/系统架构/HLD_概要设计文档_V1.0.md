## 1. 引言

### 1.1 目的
本文档基于SRS、IRS和SR需求文档，定义智能座舱主交互系统的概要设计，为详细设计和开发提供架构指导。

### 1.2 范围
涵盖系统分层架构、模块划分、接口定义、数据流、部署方案、技术选型及安全/性能架构设计。

### 1.3 设计约束
- 符合ISO 26262 ASIL B功能安全要求
- 符合ISO/SAE 21434网络安全要求
- 基于Android Automotive 12平台
- 硬件平台：高通SA8155P SoC

### 3.2 接口定义规范

#### 3.2.1 接口命名规范
- 接口格式：`I[模块名]Service`
- 方法格式：`[动作][对象]` (如：`getVehicleSpeed`)
- 事件格式：`on[事件名]` (如：`onSpeedChanged`)

#### 3.2.2 车辆服务接口 (IVehicleService)

```java
/**
 * 车辆服务接口
 * ASIL等级: ASIL B
 * 需求追溯: REQ-INT-SW-004, REQ-SAF-001-ASIL_B
 */
public interface IVehicleService {
    
    // ========== 车速相关 ==========
    
    /**
     * 获取当前车速
     * @return 车速值 (km/h)，-1表示无效
     * @throws VehicleException 通信异常
     */
    int getVehicleSpeed() throws VehicleException;
    
    /**
     * 注册车速变化监听
     * @param listener 监听器
     * @param intervalMs 最小报告间隔(ms)
     */
    void registerSpeedListener(SpeedListener listener, int intervalMs);
    
    // ========== 挡位相关 ==========
    
    /**
     * 获取当前挡位
     * @return GearPosition (P/R/N/D)
     */
    GearPosition getGearPosition();
    
    /**
     * 注册挡位变化监听
     */
    void registerGearListener(GearListener listener);
    
    // ========== 驻车制动 ==========
    
    /**
     * 获取驻车制动状态
     * @return true=启用, false=释放
     */
    boolean isParkingBrakeOn();
    
    // ========== 行驶状态(综合判断) ==========
    
    /**
     * 获取行驶限制状态
     * @return DrivingRestrictionStatus
     * 
     * 触发条件:
     * - 车速>0 OR 挡位在D/R挡 OR 驻车制动释放
     * 
     * ASIL B: 该状态采用E2E保护
     */
    DrivingRestrictionStatus getDrivingRestrictionStatus();
    
    /**
     * 注册行驶限制状态变化监听
     * 延迟要求: ≤200ms (REQ-PER-007-ASIL_B)
     */
    void registerDrivingRestrictionListener(DrivingRestrictionListener listener);
}

/**
 * 行驶限制状态
 * ASIL B级数据，带E2E校验
 */
public class DrivingRestrictionStatus {
    public boolean isRestricted;      // 是否受限
    public RestrictionReason reason;  // 限制原因
    public long timestamp;            // 时间戳
    public int e2eCounter;            // E2E序列号
    public int e2eCrc;                // E2E校验值
}
```

#### 3.2.3 消息服务接口 (IMessageService)

```java
/**
 * 消息服务接口
 * ASIL等级: ASIL A (P0消息)
 * 需求追溯: REQ-MSG-FUN-001~006
 */
public interface IMessageService {
    
    /**
     * 发送消息
     * @param msg 消息对象
     * @return 消息ID
     */
    String sendMessage(Message msg);
    
    /**
     * 获取消息列表
     * @param filter 筛选条件
     * @return 消息列表
     */
    List<Message> getMessages(MessageFilter filter);
    
    /**
     * 设置应用消息优先级
     * @param appId 应用ID
     * @param priority 优先级 (P0/P1/P2/P3)
     */
    void setAppMessagePriority(String appId, MessagePriority priority);
    
    /**
     * 注册消息监听
     */
    void registerMessageListener(MessageListener listener);
    
    /**
     * 确认消息已读
     */
    void markAsRead(String messageId);
    
    /**
     * 删除消息
     */
    void deleteMessage(String messageId);
}

/**
 * 消息对象
 */
public class Message {
    public String id;                 // 消息ID
    public String appId;              // 来源应用
    public MessagePriority priority;  // 优先级
    public String title;              // 标题
    public String content;            // 内容
    public long timestamp;            // 时间戳
    public boolean isRead;            // 是否已读
    public Bundle extras;             // 扩展数据
    public PendingIntent action;      // 点击动作
}

/**
 * 消息优先级
 */
public enum MessagePriority {
    P0_EMERGENCY,    // 紧急: 安全告警
    P1_HIGH,         // 高: 导航提示、来电
    P2_MEDIUM,       // 中: 普通通知
    P3_LOW           // 低: 资讯推送
}
```

#### 3.2.4 行驶限制服务接口 (IDrivingRestrictionService)

```java
/**
 * 行驶限制服务接口
 * ASIL等级: ASIL B
 * 需求追溯: REQ-DRV-FUN-007~010
 */
public interface IDrivingRestrictionService {
    
    /**
     * 查询应用是否受限
     * @param appId 应用ID
     * @return 限制类型
     */
    RestrictionType getAppRestriction(String appId);
    
    /**
     * 将应用加入白名单
     * @param appId 应用ID
     * @param reason 原因
     */
    void addToWhitelist(String appId, String reason);
    
    /**
     * 从白名单移除
     */
    void removeFromWhitelist(String appId);
    
    /**
     * 获取白名单列表
     */
    List<WhitelistEntry> getWhitelist();
    
    /**
     * 控制应用行为
     * @param appId 应用ID
     * @param control 控制指令
     * 
     * 响应时间要求: ≤100ms (REQ-PER-007-ASIL_B)
     */
    void controlAppBehavior(String appId, BehaviorControl control);
    
    /**
     * 注册限制状态变化监听
     */
    void registerRestrictionListener(RestrictionListener listener);
}

/**
 * 行为控制指令
 */
public enum BehaviorControl {
    PAUSE,           // 暂停
    RESUME,          // 恢复
    RETURN_HOME,     // 返回HOME
    LIMIT_INTERACTION, // 限制交互
    GRAYSCALE        // 灰度显示
}

/**
 * 限制类型
 */
public enum RestrictionType {
    NONE,            // 无限制
    VIDEO_BLOCKED,   // 视频禁止
    DOWNLOAD_BLOCKED, // 下载禁止
    GAME_BLOCKED,    // 游戏禁止
    FULL_RESTRICTED  // 完全限制
}
```

### 7.3 网络安全设计 (ISO/SAE 21434)

#### 7.3.1 网络安全控制

| 威胁 | 控制措施 | 实现方式 |
|------|----------|----------|
| 未授权访问 | 安全启动 | Secure Boot Chain |
| 数据窃听 | 传输加密 | TLS 1.3 |
| 数据篡改 | 完整性校验 | HMAC-SHA256 |
| 重放攻击 | 序列号+时间戳 | 防重放机制 |
| 恶意应用 | 签名验证 | 应用签名链验证 |
| 数据泄露 | 存储加密 | AES-256-GCM |
