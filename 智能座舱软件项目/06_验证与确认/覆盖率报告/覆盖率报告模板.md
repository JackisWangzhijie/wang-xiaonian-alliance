#### 附录C: 术语定义

| 术语 | 定义 |
|------|------|
| Line Coverage | 语句覆盖率，被测试执行到的代码行占总代码行的比例 |
| Branch Coverage | 分支覆盖率，被测试执行到的条件分支占总分支的比例 |
| Method Coverage | 方法覆盖率，被测试调用到的方法占总方法的比例 |
| Class Coverage | 类覆盖率，被测试覆盖到的类占总类的比例 |
| MC/DC | Modified Condition/Decision Coverage，修改条件/判定覆盖率 |
| ASIL | Automotive Safety Integrity Level，汽车安全完整性等级 |
