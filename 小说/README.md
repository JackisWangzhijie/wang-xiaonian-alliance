# 《都市爱情故事》

## 项目信息

- **书名**: 都市爱情故事
- **作者**: 王小年联盟
- **总章节**: 100章
- **总字数**: 约30-35万字
- **创建时间**: 2026-03-04
- **完成时间**: 2026-03-04

## 故事简介

女主严小燕，28岁广告文案，独立知性，在上海打拼。聪明、知性、独立、感情丰富、敢爱敢恨。从职场新人到创业者，从独自打拼到收获真爱，展现了现代都市女性的成长与爱情。

## 女主设定

**严小燕**
- 年龄：28岁（开篇）→ 32岁（结局）
- 职业：广告文案 → 创意总监 → 创业者
- 性格：聪明、知性、独立、感情丰富、敢爱敢恨、坚韧

## 男主设定

**沈墨白**
- 年龄：35岁
- 职业：知名建筑设计师
- 性格：外冷内热、成熟稳重、对感情认真

## 章节结构

### 第一卷：独自绽放（第001-025章）
介绍小燕在上海的生活状态，职场打拼，遇到沈墨白。

### 第二卷：心动与试探（第026-050章）
两人从陌生到熟悉，感情从暧昧到确定的过程。

### 第三卷：现实考验（第051-075章）
同居磨合，家庭压力，事业危机，共同成长。

### 第四卷：幸福定义（第076-100章）
婚礼、怀孕、生产、终章，收获爱情但不失去自我。

## 核心主题

1. **独立与亲密**：如何在爱情中保持自我
2. **事业与爱情**：不是二选一，而是相互成就
3. **现实与理想**：接受爱情的不完美，但保持对美好的追求
4. **成长与蜕变**：从28岁到32岁，小燕的成熟历程
