# A股实时行情查询工具 🇨🇳📈

基于新浪财经API的免费A股行情查询工具，**无需注册，无需Token**。

## 文件说明

| 文件 | 说明 |
|------|------|
| `cn_stock_simple.py` | 轻量版，使用新浪财经API，**无需额外依赖** ✅ |
| `cn_stock_tool.py` | 完整版，使用AKShare，功能更丰富（需安装依赖）|
| `install_stock_tool.sh` | 安装脚本 |

## 快速使用

### 1. 直接使用
```bash
# 查询单只股票
python3 cn_stock_simple.py quote 600519

# 查看热门股票
python3 cn_stock_simple.py hot

# 批量查询
python3 cn_stock_simple.py batch 600519,000001,000858
```

### 2. 使用快捷命令（推荐）
```bash
# 添加快捷命令
source ~/.bashrc

# 然后可以使用
stock quote 600519    # 查询茅台
stock hot             # 热门股票
stock batch 600519,000001  # 批量查询
```

## 支持的代码格式

- 600519 → 自动识别为沪市
- 000001 → 自动识别为深市
- 600519.SH → 标准格式
- 000001.SZ → 标准格式

## 注意事项

1. 数据仅供学习研究使用
2. 股市有风险，投资需谨慎
3. 数据延迟约3-5秒，不适合高频交易
4. 非官方接口，频繁请求可能被封IP
